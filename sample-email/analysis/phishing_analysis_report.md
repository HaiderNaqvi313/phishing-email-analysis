# Phishing Email Analysis Report

## Overview
This project analyzes a phishing email impersonating PayPal to steal user credentials.

## Indicators of Phishing
- Suspicious sender domain (paypa1.com instead of paypal.com)
- Urgent language used to pressure the user
- Malicious verification link

## Header Analysis
The sender domain appears to be a typosquatted domain designed to impersonate PayPal.

## URL Analysis
Suspicious URL found in the email:
http://paypal-verification-login.com

Threat intelligence investigation shows this domain is associated with phishing activity.

## Conclusion
The email is a credential harvesting phishing attempt designed to steal PayPal login information.
