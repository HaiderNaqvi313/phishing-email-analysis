# Phishing Email Analysis (SOC Analyst Project)

## Overview
This project demonstrates how a SOC analyst investigates a phishing email.

The analysis includes email inspection, malicious URL investigation, and extraction of Indicators of Compromise (IOCs).

## Tools Used
- VirusTotal
- Any.Run
- Email Header Analysis

## Project Structure
sample-email/
Contains the phishing email used in the investigation

analysis/
Detailed phishing investigation report

iocs/
Indicators of Compromise extracted from the email

## Skills Demonstrated
- Phishing detection
- Threat intelligence investigation
- Email header analysis
- IOC extraction
